# SinaSpider
Sina Weibo Spider in Python

## Usage

1. python spider.py
2. Input your Sina Weibo username
3. Input your Sina Weibo password
4. Input keyword
5. Input start time
6. Input end time (todo)
7. Input time interval between adjacent crawling
